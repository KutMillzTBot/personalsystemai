# ForexSmartBot Web Console

This folder is the static GitHub/phone bundle for ForexSmartBot.

Use the hosted page with a bridge query when your Cloudflare URL changes:
- `../index.html?bridge=https://YOUR-BRIDGE-URL`
- `./forexsmartbot_dashboard.html?bridge=https://YOUR-BRIDGE-URL`

The page also supports the built-in default bridge from the HTML meta tag.
